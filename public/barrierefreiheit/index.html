Barrierefreiheit

Informationen zur Barrierefreiheit dieser Website

Unser Anspruch

Wir möchten, dass diese Website für alle Menschen zugänglich ist — unabhängig von körperlichen oder kognitiven Einschränkungen, verwendetem Gerät oder Technologie.

Was wir umsetzen

  • Semantisches HTML für Screenreader-Kompatibilität
  • Ausreichende Farbkontraste (WCAG AA)
  • Tastaturnavigation für alle interaktiven Elemente
  • Textalternativen für grafische Elemente
  • Klare, verständliche Sprache

Bekannte Einschränkungen

Diese Website befindet sich im Aufbau. Einzelne Bereiche werden fortlaufend auf Barrierefreiheit geprüft und verbessert.

Feedback

Wenn du Barrieren auf dieser Website feststellst, freuen wir uns über eine Rückmeldung: borderline-verstehen@gmail.com
